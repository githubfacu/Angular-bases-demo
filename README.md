# Aplicación de Países

Single Page Application (SPA) desarrollada con **Angular**, enfocada en el consumo y manejo de información de países.

## Características
- Rutas hijas y rutas anidadas
- Creación y comunicación entre componentes
- Consumo de APIs mediante **HTTP**
- Manejo de estado y caché
- Uso de **RxJS** y operadores RxJS
- **Signals**
- Query Parameters
- Estilos con **TailwindCSS** y **DaisyUI**

## Tecnologías
- Angular
- TypeScript
- RxJS
- TailwindCSS
- DaisyUI

## Contexto
